# Getting Started with AMD GPUs for AI

This section covers the fundamentals of setting up your AMD GPU for AI workloads.

## Topics

- **ROCm Installation** - Installing AMD's ROCm platform on Linux
- **Driver Setup** - Configuring GPU drivers and permissions
- **Environment Setup** - Setting up Python environments with ROCm support
- **Verification** - Testing your GPU setup

## Supported GPUs

ROCm officially supports these GPU families:
- Radeon RX 7000 series (RDNA 3)
- Radeon RX 6000 series (RDNA 2)
- Radeon Pro W7000/W6000 series
- Instinct MI series (data center)

Check the [ROCm documentation](https://rocm.docs.amd.com/) for the latest compatibility list.
