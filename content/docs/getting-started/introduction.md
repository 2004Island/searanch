# Introduction

This is your first documentation page. 

## What is this site?

This is a collaborative documentation site where anyone can contribute content through GitHub pull requests.

## Features

- **Easy to contribute**: Just submit a PR with your markdown content
- **Beautiful theme**: Using the Hugo Book theme
- **Automatic deployment**: Changes are automatically published via GitHub Actions
- **Search enabled**: Full-text search across all documentation
